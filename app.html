Conseils chronobiologiques personnalises
🌅
Matin - Energie
Proteines, oeufs, fromage blanc, fruits rouges, avoine. Evitez les sucres rapides.
☀
Midi - Repas principal
Repas le plus complet. Proteines + legumes + feculents. Tolerance glucidique maximale.
🏦
Apres-midi - Collation
Fruits, oleagineux, yaourt. Evitez les excitants apres 15h pour preserver le sommeil.
🌙
Soir - Legerete
Repas leger 2-3h avant le coucher. Legumes, poisson blanc, tryptophane pour le sommeil.
